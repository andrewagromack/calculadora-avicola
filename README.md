# Agromack Kit Avícola

Sitio estático listo para GitHub Pages.

## Estructura
- `index.html`: landing principal con las secciones originales y formulario de contacto.
- `calculadora-costo-huevo.html`: calculadora de costo por huevo.
- `calculadora-simple.html`: calculadora avícola simple.
- `calculadora-integral-proyecto.html`: calculadora integral de proyecto.
- `assets/css/`: estilos separados por página y estilos comunes.
- `assets/js/`: scripts separados por página y navegación común.
- `assets/img/`: logo y foto de portada.

## Despliegue en GitHub Pages
1. Sube todo el contenido de esta carpeta a un repositorio.
2. En GitHub: Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: `main`, carpeta `/root`.
5. Guardar.
